სისტემური არქიტექტურა

იზოლირებული PDO ფენა

მონაცემთა ბაზასთან კომუნიკაცია ხდება ექსკლუზიურად სტანდარტიზებული PDO Wrapper-ით. სრულად გამორიცხავს SQL Injection-ის რისკებს და უზრუნველყოფს მოთხოვნების უსაფრთხო მარშრუტიზაციას.

გარე დამოკიდებულება

GoniCore დაწერილია სუფთა Native PHP-ზე. არ იყენებს მესამე მხარის ბიბლიოთეკებს, რაც ზრდის სისტემის უსაფრთხოებას.

Headless API სტანდარტი

სისტემა არ არის მიბმული კონკრეტულ Front-end ტექნოლოგიაზე. მუშაობს როგორც დამოუკიდებელი API სერვერი, რომელიც დააბრუნებს მკაცრად ფორმატირებულ JSON მონაცემებს ნებისმიერი კლიენტისთვის.
